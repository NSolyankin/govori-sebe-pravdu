# Говори себе правду

Личный дневник мотивов по второй ценности Таллинской школы менеджеров В.К. Тарасова: видеть всю комбинацию мотивов - и симпатичных, и не очень - за каждым своим поступком и за каждой оценкой чужих поступков.

Приложение: https://nsolyankin.github.io/govori-sebe-pravdu/

Пара к приложению [А что потом?](https://github.com/NSolyankin/a-chto-potom): то смотрит вперед, на последствия решения, это - назад, на мотивы поступка.

## Возможности

Записи двух типов (свой поступок / чужой поступок), мотивы с отметками "симпатичный | не очень" и "мышка", индикатор полноты из пяти шагов, наводящие вопросы "Копнуть глубже", перечитывание с поздними мотивами и отметкой честности, полка "Пора перечитать", итоги за все время и 30 дней, экспорт архива .json, записи в .md, Obsidian и печать, код на вход с Face ID.

## Приватность

Один файл index.html без сервера, сети и аналитики. Записи хранятся только в localStorage браузера на вашем устройстве. Перенос между устройствами - файлом архива .json; ссылок-снимков нет намеренно. Код на вход - заслон от случайного взгляда, настоящую защиту дает код-пароль устройства.

## Установка на iPhone

Откройте адрес в Safari, затем Поделиться - На экран "Домой". Приложение получит отдельное хранилище, не подпадающее под семидневную очистку Safari.
